Virtual CAN Automotive Simulation

A software-only simulation of an automotive CAN bus with multiple virtual ECUs and a futuristic digital instrument cluster (HMI).

Features
- Virtual CAN Bus implementation in Python
- Speed ECU, Engine ECU, Temperature ECU
- Real-time Automotive Dashboard (Tkinter GUI)
- Speedometer (0–200 km/h)
- RPM Meter (0–6000 RPM)
- Engine Temperature Monitoring
- Driving Modes: ECO / NORMAL / SPORT
- Fault Detection:
  - Overheat
  - Over-rev
  - Engine Stress
- Emergency Alert System ("SEND HELP")

Tech Stack
- Python 3
- Tkinter (GUI)
- Multithreading
- Custom CAN Bus Simulation
